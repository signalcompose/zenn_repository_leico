# 工事中
## ログ出力のレベルを設定ファイルから変更する

`Config/DefaultEngine.ini` に

```
[Core.Log]
global=[default verbosity for things not listed later]
[cat]=[level]
```

という記述をするとログ出力レベルを起動時から設定できます。
例えば

```
[Core.Log]
LogTemp=verbose
```

とすると LogTempカテゴリの表示レベルが起動時から `verbose` になります。
